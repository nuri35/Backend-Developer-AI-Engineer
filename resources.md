## Certifications with real value

### Vendor certifications (proctored exams — strongest signal)

These are real exams you take at a testing center or online with a proctor. They carry weight with employers.

| Certification | Provider | Cost | Exam | Best for |
|--------------|----------|------|------|---------|
| AWS Certified AI Practitioner (AIF-C01) | AWS | $100 | 65 questions, 90 min | AWS ecosystem, AI concepts |
| AWS Certified Generative AI Developer (AIP-C01) | AWS | ~$300 | 85 questions, 205 min | Production GenAI on AWS |
| Microsoft Azure AI App & Agent Developer (AI-103) | Microsoft | ~$165 | Multiple choice | Azure OpenAI, agents |
| NVIDIA Certified Associate: Generative AI & LLMs (NCA-GENL) | NVIDIA | $125 | 50-60 questions, 60 min | LLM concepts, RAG basics |

**Recommendation:** Start with NVIDIA NCA-GENL ($125) — it's available now, affordable, and covers the fundamentals. Then get AWS AIP-C01 or Microsoft AI-103 once they are fully available.

### Coursera professional certificates (completion-based — good for learning)

These require passing quizzes and completing projects. They give you a Credly badge you can add to LinkedIn.

| Certificate | Provider | Time | Best for |
|------------|----------|------|---------|
| IBM RAG and Agentic AI Professional Certificate | IBM | 3 months | RAG, LangChain, LangGraph, MCP |
| Microsoft Generative AI Engineering | Microsoft | 3 months | Azure, RAG, deployment |
| AWS GenAI and AI Agents with Amazon Bedrock | AWS | 2 months | Bedrock, RAG, agents |
| Generative AI for Software Development | DeepLearning.AI | 2 months | Dev workflow, prompts |
| IBM Generative AI Engineering | IBM | 6 months | Full pipeline, deep coverage |

All are accessible through Coursera Plus at ~$49/month.

### DataCamp

Has a real proctored certification (2-hour exam + 4-hour practical).

| Track | Time | Topics |
|-------|------|--------|
| Associate AI Engineer for Developers | 26 hours | OpenAI API, LangChain, Pinecone, LLMOps |

## Recommended learning order

**Month 1 (free):**
1. DeepLearning.AI: ChatGPT Prompt Engineering for Developers
2. DeepLearning.AI: LangChain: Chat with Your Data
3. NVIDIA DLI: Building RAG Agents with LLMs
4. DeepLearning.AI: Functions, Tools and Agents with LangChain

**Month 2 (free + optional paid):**
5. DeepLearning.AI: AI Agents in LangGraph
6. Hugging Face AI Agents Course
7. LangChain Academy
8. Optional: Udemy course by Ed Donner (~$15)

**Month 3 (certification):**
9. NVIDIA NCA-GENL exam ($125)
10. Start AWS AIF-C01 or IBM RAG certificate on Coursera
